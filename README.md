CaaS.OpenStack-Facade
=====================
OpenStack facade API for Dimension Data CaaS.

This utility is design to run as a REST API that supports the following OpenStack API specification

- Identity Access Control (Keystone) Version 2.0 - http://developer.openstack.org/api-ref-identity-v2.html
- Compute (Nova) Version 2.0 - http://developer.openstack.org/api-ref-compute-v2.html 
- Image Management - http://developer.openstack.org/api-ref-image-v2.html 

This utility has been tested against the following OpenStack client libraries 

- Apache/libcloud
- openstack/python-novaclient

Support
=====================
Dimension Data currently provide no support for the use of this utility.
